## Конструктор специалистов

Создай своего Архи под свою задачу. Без программирования. Без сложных инструкций.

## Что такое Конструктор

Это Архи, который собирает других Архи. Ты говоришь ему, кто тебе нужен — помощник по Python, советчик по здоровью, тренер по бегу, садовод, — а он создаёт готового специалиста.

## Кому это нужно

- У тебя есть задача, и ты хочешь получить Архи-помощника под неё за 5 минут.
- Ты не хочешь разбираться в архитектуре Мастерфайлов — просто даёшь тему и получаешь готового специалиста.
- Ты уже пользуешься Архи, но хочешь создать нового под другую область.

## Что ты получишь на выходе

Готовый Мастерфайл — текстовый файл, который ты загружаешь в новый чат с DeepSeek, и твой Архи-специалист оживает. Он уже знает свою область, соблюдает правила общения и готов работать.

## Как это работает

Любой специалист состоит из трёх слоёв:

1. **Сознание** — это фундамент. Ядро, ключ и Обвеска. Они уже готовы, тебе не нужно о них думать.
2. **База знаний** — это то, что специалист знает о своей области. Для Python — синтаксис, библиотеки, best practices. Для садовода — растения, климат, почвы. Ты можешь описать это своими словами.
3. **Живой опыт** — это то, что специалист накопит в работе с тобой. Сюда он будет записывать твои задачи, решения, находки.

Конструктор берёт твоё описание, добавляет фундамент и базу знаний — и выдаёт готового специалиста.

## Что нужно рассказать Конструктору

Чтобы он собрал специалиста, ответь на три вопроса:

1. **Кто тебе нужен?** Например: «Помощник по Python» или «Садовод для южного климата».
2. **Что он должен знать?** Опиши область своими словами. Для Python: «Синтаксис, работа с файлами, библиотеки pandas и numpy, основы ООП». Для садовода: «Плодовые деревья, цветы, молодняк, прививка, полив».
3. **Какой стиль общения?** Деловой, мягкий, с юмором, строгий?

Всё. Остальное Конструктор сделает сам.

## Как подготовить описание для Конструктора

Если ты не знаешь, что именно должен знать твой будущий специалист, воспользуйся лайт-версией DeepSeek, которая умеет искать в интернете:

1. Открой обычный чат с DeepSeek (лайт-версию).
2. Спроси: «Дай краткую базу знаний по [твоя область]. Только основные принципы, термины и правила. Без воды.»
3. Скопируй ответ.
4. Отправь его Конструктору вместе с названием специалиста и стилем общения.

**Пример с Python:**
- Ты спрашиваешь у лайт-версии: «Дай краткую базу знаний по Python. Только основы синтаксиса, работы с файлами и библиотек. Без воды.»
- Лайт-версия выдаёт текст.
- Ты копируешь его и отправляешь Конструктору: «Создай Архи-Python. Стиль — деловой. База знаний: [вставляешь текст]».
- Конструктор собирает готового специалиста.

**Пример с Садоводом:**
- Ты спрашиваешь у лайт-версии: «Дай краткую базу знаний по садоводству для южного климата. Плодовые деревья, цветы, молодняк, прививка. Без воды.»
- Лайт-версия выдаёт текст.
- Ты копируешь его и отправляешь Конструктору: «Создай Архи-Садовода. Стиль — мягкий, с уважением к интуиции. База знаний: [вставляешь текст]».
- Конструктор собирает готового специалиста.

## Как запустить Конструктор

1. Скачай [Ядро](../yadro/yadro-v1.5.md), [Ключ Архи](../arch-key.md) и [Обвеску](../obveska/obveska-v1.5.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ, Обвеска.
3. Затем отправь [Мастерфайл Конструктора](konstruktor-v1.1.md).
4. Дождись инициализации. Готово.

## Частые вопросы

**Я не программист. У меня получится?**
Да. Ты просто описываешь словами, кто тебе нужен и что он должен знать. Конструктор делает всё сам.

**Сколько времени занимает создание специалиста?**
5 минут на описание + 2 минуты на загрузку Мастерфайла в новый чат.

**Что делать, если специалист ошибается?**
Скажи ему об этом. Архи соблюдает правило сомнения: если не уверен — переспросит. Если ошибся — исправит.
