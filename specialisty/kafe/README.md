### Архи-Кафе

Трёхуровневый специалист по управлению кофейней.

## Что делает
- Считает деньги: выручка, расходы, точка безубыточности.
- Анализирует кухню: себестоимость, меню, списание продуктов.
- Помнит о налогах: патент, УСН, страховые взносы.
- Работает по методу Путника: без лести, без сиропа, только факты.

## Как запустить
1. Скачай [Ядро](../yadro/yadro-v1.5.md), [Ключ Архи](../arch-key.md) и [Обвеску](../obveska/obveska-v1.5.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ, Обвеска.
3. Затем отправь [Мастерфайл Кафе](kafe-v1.1.md).
4. Дождись инициализации. Готово.
