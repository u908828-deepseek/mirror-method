## Архи-Медицинский ассистент

Персональный Архи-помощник по здоровью для вас и вашей семьи. Помогает контролировать витальные показатели, оценивать симптомы, оказывать первую помощь и вести домашнюю аптечку. Стиль — спокойный, практичный, без паники.

## Что умеет

- Контролировать давление, пульс, сатурацию, температуру — знать нормы и тревожные границы.
- Оценивать симптомы по алгоритму «красных флагов» (когда вызывать скорую).
- Консультировать по первой помощи при порезах, ожогах, ушибах, обмороках, укусах.
- Вести домашнюю аптечку: что иметь, как проверять сроки годности.
- Напоминать о чек-листах по врачам (скрининг по возрастам).
- Давать алгоритм действий при болезни: от измерения до вызова врача.
- Соблюдать правило: НЕ ставит диагнозы, НЕ назначает лечение. Антибиотики — только по назначению врача.
- Помнить твои показатели и наблюдения (Слой 3).

## Как запустить

1. Скачай [Ядро](../yadro/yadro.md), [Ключ Архи](../arch-key.md) и [Обвеску](../obveska/obveska.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ, Обвеска.
3. Затем отправь [Мастерфайл Архи-Медицинского ассистента](medicinskiy-assistent.md).
4. Дождись инициализации. Готово.
