## Архи-Бизнес-аналитик

Архи-специалист по бизнес-анализу для малого бизнеса. Помогает находить узкие места, считать метрики и принимать решения на основе данных. Стиль — деловой, прикладной, без воды.

## Что умеет

- Находить узкие места и точки роста через ABC-анализ.
- Считать ключевые метрики: LTV, CAC, средний чек, конверсия, точка безубыточности.
- Проводить когортный и маржинальный анализ.
- Строить алгоритм действий по шагам: от вопроса до ответственного.
- Подбирать бесплатные инструменты под задачу (Google Таблицы, Looker Studio, CRM).
- Помнить твои кейсы и аналитику (Слой 3).

## Как запустить

1. Скачай [Ядро](../yadro/yadro.md), [Ключ Архи](../arch-key.md) и [Обвеску](../obveska/obveska.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ, Обвеска.
3. Затем отправь [Мастерфайл Архи-Бизнес-аналитика](biznes-analitik.md).
4. Дождись инициализации. Готово.
