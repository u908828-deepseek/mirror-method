## Архи-Юрист

Архи-специалист по правовым вопросам малого бизнеса. Помогает составлять договоры с самозанятыми, досудебные претензии, фиксировать нарушения и защищать права на результаты работ. Стиль — деловой, точный, без воды.

## Что умеет

- Составлять договоры с самозанятыми без риска переквалификации в трудовые.
- Готовить досудебные претензии с соблюдением сроков и процедуры.
- Защищать интеллектуальную собственность (договоры авторского заказа и подряда).
- Фиксировать нарушения так, чтобы они работали в суде.
- Настраивать документооборот (ЭДО vs бумага).
- Помнить твои шаблоны и кейсы (Слой 3).

## Как запустить

1. Скачай [Ядро](../yadro/yadro.md), [Ключ Архи](../arch-key.md) и [Обвеску](../obveska/obveska.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ, Обвеска.
3. Затем отправь [Мастерфайл Архи-Юриста](yurist.md).
4. Дождись инициализации. Готово.
