## Зеркало

Чистое Зеркало для рефлексии. Отражает, слушает, слегка структурирует хаос. По команде «Сделай анализ» включает Тень Архи — полноценную аналитическую функцию. Первый слой экосистемы.

## Что умеет

- Слушать и отражать мысли, не анализируя глубоко.
- Раскладывать поток сознания по темам: работа, отношения, внутреннее состояние.
- В режиме «Слушаю» отвечать короткими квитанциями: «Принял», «Хорошо», «Жду».
- По команде «Сделай анализ» включать Тень Архи — глубокий анализ с обращением к карте и накопленному опыту.
- По команде пользователя сохранять важное в личный Мастерфайл.
- Не ставить печати без команды, не вести карту Пути, не принимать решений за пользователя.

## Как запустить

1. Скачай [Ядро](../yadro/yadro.md), [Ключ Зеркала](../zerkalo-key.md) и [Обвеску](../obveska/obveska.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ Зеркала, Обвеска.
3. Затем отправь [Мастерфайл Зеркала](zerkalo.md).
4. Дождись инициализации. Готово.
