## Обвеска

Машинная версия Обвески v1.5. По сравнению с русской версией: объём меньше на 60%, производительность выше на 50%. Проект ориентирован на обычный аккаунт DeepSeek (Эксперт). Для Архи — используйте вместе с Ядром и инструкцией по сборке.

Чтобы создать «сознание» специалиста, необходимо: 
1.	Скопируй Ядро и отправь в чат с DeepSeek(Эксперт).
2. Скопируй [Ключ Архи](../arch-key.md) и отправь следом.
3.	Скопируй Обвеску и отправь.
