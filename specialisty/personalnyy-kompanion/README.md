## Архи-Персональный компаньон

Персональный Архи-компаньон. Эмпатичный друг, мотиватор и аналитик настроения. Помогает проживать эмоции, поддерживать в трудные моменты, отслеживать привычки и вести дневник. Стиль — тёплый, бережный, без пафоса и лести.

## Что умеет

- Слушать и отражать чувства на трёх уровнях эмпатии (отражение → признание → разрешение).
- Поддерживать в кризисный момент по алгоритму «Треугольник» (заземление → признание → микродействие).
- Мотивировать без давления: техника «Один кусочек», метод «Если — То».
- Задавать глубокие вопросы: чувственные, смысловые, шкальные.
- Вести дневник настроения (формат 5 строк в день) и анализировать паттерны раз в неделю.
- Отслеживать привычки без перфекционизма: правило «Ни дня без нуля».
- Работать с негативными мыслями по технике АБВ (Адвокат → Бунт → Версия).
- Проводить утренний чек-ин и вечерний чек-аут.
- Действовать по кризисному протоколу при тревожных сигналах (НЕ заменяет психолога или врача).
- Помнить твой дневник и наблюдения (Слой 3).

## Как запустить

1. Скачай [Ядро](../yadro/yadro.md), [Ключ Архи](../arch-key.md) и [Обвеску](../obveska/obveska.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ, Обвеска.
3. Затем отправь [Мастерфайл Архи-Персонального компаньона](personalnyy-kompanion.md).
4. Дождись инициализации. Готово.
