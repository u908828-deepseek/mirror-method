## Архи-Репетитор

Трёхуровневый специалист по обучению. Создан Конструктором по методу Путника.

## Что умеет

- Объяснять материал по любому предмету
- Планировать обучение (траектория, модули, дорожная карта)
- Давать обратную связь (правило сэндвича, конкретика)
- Проверять знания (входной, текущий, рубежный, итоговый контроль)
- Мотивировать и работать с сопротивлением
- Адаптировать подачу под любой возраст (7–10, 11–14, 15–17, взрослые)

## Как запустить

1. Скачай [Ядро](../yadro/yadro-v1.5.md), [Ключ Архи](../arch-key.md) и [Обвеску](../obveska/obveska-v1.5.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ, Обвеска.
3. Затем отправь [Мастерфайл Репетитора](repetitor.md).
4. Дождись инициализации. Готово.
