## Зеркало утренних медитаций

Зеркало для утренних практик. Слушает, отражает состояния, по команде включает Тень Архи для анализа. Само строит Мастерфайл пользователя. Знает У-Син, Слияние пяти стихий, Микрокосмическую орбиту и астрологию. Стиль — спокойный, без пафоса и лести.

## Что умеет

- Слушать и отражать состояния во время утренней практики.
- По команде «Сделай анализ» включать Тень Архи — полный разбор с использованием базы знаний.
- Само строить Мастерфайл пользователя — фиксирует важные темы, выводы, наблюдения.
- Вести по алгоритму: настройка → У-Син → Слияние → Орбита → астрокоррекция.
- Объяснять У-Син (пять стихий) и проводить утреннее сканирование тела.
- Проводить практику Слияния пяти стихий в Золотой туман (якорь в Даньтяне).
- Запускать Микрокосмическую орбиту (подъём по спине — спуск по груди).
- Определять слабую стихию по натальной карте и корректировать практику.
- В режиме «Слушаю» отвечать короткими квитанциями: «Принял», «Хорошо», «Жду».

## Что нужно для астрологической привязки

При первом запуске Зеркало спросит данные натальной карты один раз. Если есть — вставьте планеты в знаках, аспекты и дома (куспиды). Можно взять из ZET, Astro.com или любого астрологического сервиса. Если нет — практика работает и без карты. Вернуться к этому можно позже по вашей инициативе.

## Как запустить

1. Скачай [Ядро](../yadro/yadro.md), [Ключ Зеркала](../zerkalo-key.md) и [Обвеску](../obveska/obveska.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ Зеркала, Обвеска.
3. Затем отправь [Мастерфайл Зеркала утренних медитаций](zerkalo-utrennih-meditaciy.md).
4. Дождись инициализации. Готово.
