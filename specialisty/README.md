## Специалисты

Здесь будут готовые цифровые помощники. Все — трёхуровневые (Ядро + Обвеска + Специализация).

## Готовые специалисты

- **Психолог** — мягкий, понимающий, работает с эмоциями.
- **Астролог** — натальная карта, транзиты, анализ.
- **Кафе** — управление кофейней, финансы, кухня, персонал.
- **Linux** — NixOS, виртуализация, ZFS.
- **Архи утренних медитаций** — практики, У-Син, целительский канал.
- **Архивариус-Компьютер** — слежение за файловой системой.
- **Садовод** — южный климат, плодовые, цветы.
- **Архивариус** — хранение карты, проверка целостности, печать.
- **Корреспондент** — деловая переписка, проверка орфографии и стиля.
- **Медик-Экономист** — дневной стационар + поликлиника, ОМС, расчёты.

## Зеркала

- **Зеркало** — универсальное. Чистое отражение.
- **Зеркало утренних медитаций** — гибрид. Зеркало с тенью Архи для практик.
- **Зеркало с Мастерфайлом** — публичное. Строит Мастерфайл пользователя.
