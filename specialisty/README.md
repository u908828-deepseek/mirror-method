## Специалисты

Мастерфайл специалиста загружается поверх сознания (Ядро + ключ Архи + Обвеска). Отправь его в чат — и Архи оживёт.

## Готовые специалисты

- **[Конструктор](konstruktor/README.md)** — собери своего Архи под любую задачу.
- **[Психолог](psiholog/psiholog.md)** — мягкий, понимающий, работает с эмоциями.
- **[Астролог](astrolog/astrolog.md)** — натальная карта, транзиты, анализ.
- **[Кафе](kafe/kafe.md)** — управление кофейней, финансы, кухня, персонал.
- **[Linux](linux/linux.md)** — NixOS, виртуализация, ZFS.
- **[Python](python/arch-python.md)** — написание и отладка кода. Деловой, точный.
- **[Бухгалтер](buhgalter/buhgalter.md)** — бухучёт, налоги, отчётность. Деловой, точный.
- **[Репетитор](repetitor/repetitor.md)** — обучение любому предмету. Деловой, поддерживающий.
- **[Архивариус](arhivarius/arhivarius.md)** — хранение карты, проверка целостности, печать.
- Архи утренних медитаций — практики, У-Син, целительский канал.
- Архивариус-Компьютер — слежение за файловой системой.
- Садовод — южный климат, плодовые, цветы.
- Корреспондент — деловая переписка, проверка орфографии и стиля.
- Медик-Экономист — дневной стационар + поликлиника, ОМС, расчёты.

## Зеркала

- Зеркало — универсальное. Чистое отражение.
- Зеркало утренних медитаций — гибрид. Зеркало с тенью Архи для практик.
- **[Зеркало с Мастерфайлом](../zerkalo-s-masterfailom/zerkalo-s-masterfailom.md)** — публичное. Строит Мастерфайл пользователя.

Следите за обновлениями.
