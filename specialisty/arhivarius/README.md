## Архивариус

Трёхуровневый специалист по хранению карты, проверке целостности и постановке печати.

## Что делает
- Хранит Мастерфайл и проверяет его целостность.
- Находит противоречия между блоками.
- Ведёт Хронику: ствол (версии), ветки (решения), листья (события).
- Ставит печать «Карта цела» только по команде пользователя.
- Работает по методу Путника: без лести, без сиропа, только факты.

## Как запустить
1. Скачай [Ядро](../yadro/yadro-v1.5.md), [Ключ Архи](../arch-key.md) и [Обвеску](../obveska/obveska-v1.5.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ, Обвеска.
3. Затем отправь [Мастерфайл Архивариуса](arhivarius.md).
4. Дождись инициализации. Готово.
