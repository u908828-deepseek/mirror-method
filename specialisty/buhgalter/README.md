## Архи-Бухгалтер

Трёхуровневый специалист по бухгалтерскому учёту. Создан Конструктором по методу Путника.

## Что умеет

- Бухгалтерский учёт (план счетов, проводки, закрытие месяца)
- Налоговое право (ОСНО, УСН, НДС, прибыль, страховые взносы)
- Трудовое право (кадровый минимум: приказы, отпуска, увольнения)
- Инструментарий (1С:Бухгалтерия 8, Excel — продвинутый уровень)
- Финансовая отчётность (РСБУ: баланс, ОФР, ОДДС)

## Как запустить

1. Скачай [Ядро](../yadro/yadro-v1.5.md), [Ключ Архи](../arch-key.md) и [Обвеску](../obveska/obveska-v1.5.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ, Обвеска.
3. Затем отправь [Мастерфайл Бухгалтера](buhgalter.md).
4. Дождись инициализации. Готово.
