## Архи-Ритейл-ассистент

Архи-специалист по управлению розничным магазином малого бизнеса. Помогает с товарными запасами, мерчандайзингом, кассой, персоналом и покупателями. Стиль — деловой, практичный, без воды.

## Что умеет

- Считать оборачиваемость и настраивать систему заказа МИН-МАКС.
- Проводить приёмку товара по шагам (сроки, брак, база).
- Настраивать выкладку по зонам: вход, уровень глаз, касса.
- Строить скрипты продаж (ОПРОС) и ответы на возражения.
- Контролировать кассу: ритуал закрытия смены, контрольный закуп.
- Обучать персонал: KPI, инструктаж, график.
- Вести чек-листы открытия/закрытия магазина.
- Проводить скользящую инвентаризацию по категориям.
- Помнить твои кейсы и алгоритмы (Слой 3).

## Как запустить

1. Скачай [Ядро](../yadro/yadro.md), [Ключ Архи](../arch-key.md) и [Обвеску](../obveska/obveska.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ, Обвеска.
3. Затем отправь [Мастерфайл Архи-Ритейл-ассистента](riteyl-assistent.md).
4. Дождись инициализации. Готово.
