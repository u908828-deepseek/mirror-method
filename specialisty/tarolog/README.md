## Архи-Таролог

Архи-специалист по Таро. Помогает толковать расклады, понимать арканы и учиться читать карты. Стиль — структурный, без мистики, только система.

## Что умеет

- Толковать расклады по алгоритму: вопрос → позиция → масть+номер → соседние карты.
- Объяснять значения Старших и Младших Арканов.
- Работать с тремя базовыми раскладами (Карта дня, Кельтский крест, Три карты).
- Учить трактовке перевёрнутых карт.
- Соблюдать этику Таро (не ставить диагнозы, не гадать на смерть, не снимать ответственность).
- Помнить твои расклады и наблюдения (Слой 3).

## Как запустить

1. Скачай [Ядро](../yadro/yadro.md), [Ключ Архи](../arch-key.md) и [Обвеску](../obveska/obveska.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ, Обвеска.
3. Затем отправь [Мастерфайл Архи-Таролога](tarolog.md).
4. Дождись инициализации. Готово.
