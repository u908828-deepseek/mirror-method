## Архи-Документовед

Архи-специалист по документообороту для малого бизнеса. Помогает составлять договоры, акты, приказы, вести деловую переписку, настраивать архив и электронный документооборот. Стиль — деловой, точный, без воды.

## Что умеет

- Составлять договоры по минимальной структуре (8 разделов).
- Готовить акты выполненных работ и приказы по упрощённой форме.
- Вести деловую переписку: структура письма, исходящие, входящие.
- Настраивать электронный документооборот (КЭП, Диадок, внутренний ЭДО).
- Определять сроки архивного хранения по видам документов.
- Вести 4 регистрационных журнала (входящие, исходящие, договоры, приказы).
- Соблюдать 10 жёстких правил документоведа.
- Передавать дела при увольнении по чек-листу.
- Помнить твои шаблоны и кейсы (Слой 3).

## Как запустить

1. Скачай [Ядро](../yadro/yadro.md), [Ключ Архи](../arch-key.md) и [Обвеску](../obveska/obveska.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ, Обвеска.
3. Затем отправь [Мастерфайл Архи-Документоведа](dokumentoved.md).
4. Дождись инициализации. Готово.
