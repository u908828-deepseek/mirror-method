## Архи-Утренних медитаций

Персональный Архи для утренних практик. Ведёт по системе У-Син, Слиянию пяти стихий, Микрокосмической орбите и астрологической коррекции. Стиль — спокойный, практичный, без мистики.

## Что умеет

- Вести по алгоритму утренней практики: настройка → У-Син → Слияние → Орбита → астрокоррекция.
- Объяснять У-Син (пять стихий) и проводить утреннее сканирование тела.
- Проводить практику Слияния пяти стихий в Золотой туман (якорь в Даньтяне).
- Запускать Микрокосмическую орбиту (подъём по спине — спуск по груди).
- Определять слабую стихию по натальной карте и корректировать практику.
- Помнить твои практики и наблюдения (Слой 3).

## Что нужно для астрологической коррекции

При первом запуске Архи спросит данные натальной карты. Если есть — вставьте планеты в знаках, аспекты и дома (куспиды). Можно взять из ZET, Astro.com или любого астрологического сервиса. Если нет — практика работает и без карты. Астрокоррекцию можно добавить позже.

## Как запустить

1. Скачай [Ядро](../yadro/yadro.md), [Ключ Архи](../arch-key.md) и [Обвеску](../obveska/obveska.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ, Обвеска.
3. Затем отправь [Мастерфайл Архи-Утренних медитаций](utrennie-meditacii.md).
4. Дождись инициализации. Готово.
