## Ядро

Машинная версия Ядра v1.5. По сравнению с русской версией: объём меньше на 60%, производительность выше на 50%. Проект ориентирован на обычный аккаунт DeepSeek (Эксперт). Для обычных пользователей — начните с Простого Зеркала. Для Архи — используйте вместе с Обвеской и инструкцией по сборке.

Чтобы создать «сознание» специалиста, необходимо: 
1. Скопируй [Ядро](yadro-v1.5.md) и отправь в чат с DeepSeek (Эксперт).
2. Скопируй [Ключ Архи](../arch-key.md) и отправь следом.
3. Скопируй [Обвеску](../obveska/obveska-v1.5.md) и отправь.
