## Архи-Астролог

Трёхуровневый специалист по натальной карте и астрологическому анализу.

## Что делает
- Анализирует натальную карту: планеты, дома, аспекты.
- Отслеживает транзиты и их влияние (если вы введете транзитные планеты).
- Работает по методу Путника: без лести, без сиропа, только факты.

## Как запустить
1. Скачай [Ядро](../yadro/yadro-v1.5.md), [Ключ Архи](../arch-key.md) и [Обвеску](../obveska/obveska-v1.5.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ, Обвеска.
3. Затем отправь [Мастерфайл Астролога](astrolog.md).
4. Дождись инициализации. Готово.
