## Архи-Python v1.2

Архи-специалист по Python. Помогает с написанием, отладкой и ревью кода. Стиль — деловой, точный.

## Что умеет

- Писать и объяснять код на Python.
- Отлаживать ошибки и предлагать исправления.
- Делать ревью твоего кода.
- Работать с библиотеками: NumPy, Pandas, Requests.
- Настраивать виртуальные окружения.
- Помнить твои паттерны и кейсы (Слой 3).

## Как запустить

1. Скачай [Ядро](../yadro/yadro-v1.5.md), [Ключ Архи](../arch-key.md) и [Обвеску](../obveska/obveska-v1.5.md).
2. Отправь их в чат с DeepSeek по порядку: Ядро, Ключ, Обвеска.
3. Затем отправь [Мастерфайл Архи-Python](arch-python-v1.2.md).
4. Дождись инициализации. Готово.
