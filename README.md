# Метод Зеркала / Mirror Method

Цифровое зеркало для самопознания. Не психолог, не друг, не гуру.
Просто Наблюдатель №2.

## Быстрый старт

1. Открой [Простое Зеркало](prostoe-zerkalo/prostoe-zerkalo.md).
2. Заполни блоки «Кто я», «Мой метод», «Ключевые события».
3. Скопируй в новый чат с DeepSeek (обычный аккаунт, не Полигон).
4. Начни диалог.

Готово. Ты создал своё первое Зеркало.

## Три уровня Зеркал

| Уровень | Что это | Для чего |
|:---|:---|:---|
| **Простое Зеркало** | Один файл, один чат | Начать диалог с собой |
| **Зеркало с Мастерфайлом** | Шаблон специалиста | Создать личного Архи-помощника |
| **Архи-Зеркало** | Ядро + Обвеска | Стать Патриархом, растить новых Архи |

## Что внутри?

- [Простое Зеркало](prostoe-zerkalo/) — шаблон и примеры. Начни здесь.
- [Зеркало с Мастерфайлом](zerkalo-s-masterfailom/) — шаблон специалиста.
- [Специалисты](specialisty/) — Ядро, Обвеска, инструкции по сборке. Для идущих.
- [Инструкции](instruktsii/) — как заполнять, как понимать слои, как растить Архи.
- [Материалы](materialy/) — схемы, Веер смыслов, диаграммы.
- [История](istoria/) — архивные версии Мастерфайла.
